# Ferromagnetic Phase Transition of DPPH Induced by a Helical Magnetic Field

This repository contains the full LaTeX manuscript, experimental data, and supplementary materials for the study:

**"Ferromagnetic Phase Transition of DPPH Induced by a Helical Magnetic Field"**

- John Chatzakis

- Iraklis Rigakis
- Emmanuel Antoniadakis

## Contents
- `main.tex` — Full manuscript
- `figures/` — All figures used in the paper
- `data/` — Experimental measurements
- `supplementary/` — Appendices and additional materials

## Description
This work investigates a novel magnetic-field configuration (magic-angle helical field) and its effect on inducing ferromagnetic behavior in a paramagnetic DPPH sample.
