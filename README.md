## Description

This work presents a novel experimental configuration using a **magic-angle helical magnetic field** (~54.74°) and demonstrates an anomalous magnetization response in a DPPH sample, suggesting a transition from paramagnetic to weakly ferromagnetic behavior at room temperature.

The repository provides:
- Full LaTeX source for reproducibility
- Experimental datasets
- Supplementary derivations and appendices
- Figures used in the manuscript

---
